# 中后台管理系统技术方案

## 一、技术选型

### 后端技术栈

| 技术     | 选择            | 说明                 |
| -------- | --------------- | -------------------- |
| 编程语言 | Go 1.21+        | 高性能、并发能力强   |
| Web框架  | Beego 2.0       | 完整MVC架构、内置ORM |
| ORM      | Beego ORM       | Beego内置ORM框架     |
| 数据库   | MySQL 8.0       | 关系型数据库         |
| 缓存     | Redis           | Token缓存、会话管理  |
| 认证     | JWT + OAuth 2.0 | 无状态认证           |
| API文档  | Swagger         | 自动生成API文档      |
| 配置管理 | Beego Config    | Beego内置配置管理    |

### 前端技术栈

| 技术       | 选择         | 说明                 |
| ---------- | ------------ | -------------------- |
| 框架       | Vue 3.4+     | 组合式API            |
| UI组件库   | Element Plus | Vue 3官方推荐        |
| 状态管理   | Pinia        | Vue 3推荐状态管理    |
| 构建工具   | Vite         | 开发体验好、热更新快 |
| HTTP客户端 | Axios        | 主流HTTP请求库       |
| 路由       | Vue Router 4 | 官方路由             |
| TypeScript | 5.0+         | 类型安全             |

## 二、系统架构设计

```
┌─────────────────────────────────────────────────────────┐
│                    前端 (Vue3 + Element Plus)            │
│  部门管理 │ 菜单管理 │ 角色管理 │ 用户管理 │ 岗位管理 │ 字典管理 │
└────────────────────────┬────────────────────────────────┘
                         │ HTTP/JSON
┌────────────────────────▼────────────────────────────────┐
│                    API网关/负载均衡 (Nginx)               │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│              Go后端 API服务 (Beego 2.0)                  │
│  ┌──────────────────────────────────────────────────┐   │
│  │              核心模块 (7个)                        │   │
│  │  认证 │ 部门 │ 菜单 │ 角色 │ 用户 │ 岗位 │ 字典   │   │
│  └──────────────────────────────────────────────────┘   │
│  ┌──────────────────────────────────────────────────┐   │
│  │              基础设施层                           │   │
│  │     数据库(MySQL)  │  缓存(Redis)  │  日志       │   │
│  └──────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

## 五、项目目录结构（Beego MVC架构）

```
caelum/
├── backend/                    # Go后端 (Beego)
│   ├── conf/                   # 配置文件
│   ├── controllers/           # 控制器层
│   ├── models/                # 模型层
│   ├── routers/               # 路由配置
│   ├── services/              # 业务逻辑层
│   ├── utils/                 # 工具函数
│   ├── middleware/            # 中间件
│   ├── docs/                  # Swagger文档
│   ├── main.go                # 入口文件
│   └── go.mod
│
├── frontend/                   # Vue3前端
│   ├── public/                # 静态资源
│   ├── src/
│   │   ├── api/               # API接口
│   │   ├── assets/            # 资源文件
│   │   ├── components/        # 公共组件
│   │   ├── directive/        # 自定义指令
│   │   ├── layouts/          # 布局组件
│   │   ├── router/           # 路由配置
│   │   ├── stores/           # 状态管理
│   │   ├── utils/            # 工具函数
│   │   ├── views/            # 页面组件
│   │   ├── App.vue
│   │   └── main.ts
│   ├── index.html
│   ├── package.json
│   └── vite.config.ts
│
├── docs/                       # 文档
│   ├── database.sql           # 数据库脚本
│   └── api.md                 # API文档
│
└── deploy/                    # 部署配置
    ├── docker/
    └── nginx/
```

## 六、Beego项目结构详解

### 后端目录说明

```
backend/
├── conf/                       # 配置文件目录
│   └── app.conf               # Beego主配置文件
│
├── controllers/               # 控制器层（处理请求）
│   ├── base.go               # 基础控制器
│   ├── auth_controller.go    # 认证相关
│   ├── user_controller.go    # 用户管理
│   ├── dept_controller.go    # 部门管理
│   ├── menu_controller.go    # 菜单管理
│   ├── role_controller.go    # 角色管理
│   ├── post_controller.go    # 岗位管理
│   └── dict_controller.go    # 字典管理
│
├── models/                    # 模型层（数据模型）
│   ├── user.go               # 用户模型
│   ├── dept.go               # 部门模型
│   ├── menu.go               # 菜单模型
│   ├── role.go               # 角色模型
│   ├── post.go               # 岗位模型
│   └── dict.go               # 字典模型
│
├── services/                  # 业务逻辑层
│   ├── auth_service.go       # 认证服务
│   ├── user_service.go       # 用户服务
│   ├── dept_service.go       # 部门服务
│   ├── menu_service.go       # 菜单服务
│   ├── role_service.go       # 角色服务
│   ├── post_service.go       # 岗位服务
│   └── dict_service.go       # 字典服务
│
├── routers/                   # 路由配置
│   └── router.go             # 路由注册
│
├── middleware/                # 中间件
│   ├── auth.go               # 认证中间件
│   ├── cors.go               # 跨域中间件
│   └── logging.go            # 日志中间件
│
├── utils/                     # 工具函数
│   ├── jwt.go                # JWT工具
│   ├── password.go           # 密码加密
│   └── response.go           # 响应封装
│
├── docs/                      # Swagger文档
│
├── main.go                    # 程序入口
└── go.mod                     # Go模块依赖
```
